# Magic Fold

A web-based zine maker that lets you create 8-panel mini magazines from a single sheet of paper.

**Try it out:** [magic-fold.vercel.app](https://magic-fold.vercel.app/)

## What's a Zine?

A zine is a small self-published booklet made from one sheet of paper with a single cut and a few folds. The app includes folding instructions to help you assemble your creation.

## Features

- Draw, add text, and insert images on each panel
- Export as print-ready PDF with fold/cut guides
- Share your zine via URL
- Flip book preview to see how your zine will look

## Tech Stack

- Next.js
- Fabric.js (canvas editing)
- jsPDF (PDF generation)
- Tailwind CSS
